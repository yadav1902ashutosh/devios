How to Display HTML Code as Text in a Webpage

When writing a blog or documentation about web development, you might want to display HTML code as plain text rather than letting the browser interpret it as actual HTML. In this article, we'll explore different ways to achieve this.

Why Does HTML Render Instead of Displaying as Text?

Browsers are designed to interpret HTML tags and render them accordingly. For example, writing: 

            
This is a paragraph.
Will result in the browser displaying: This is a paragraph. Instead of the actual HTML code. To prevent this, we need to explicitly tell the browser to treat HTML as plain text.

Methods to Display HTML Code as Text

1. Using HTML Entities

One simple way to display HTML code is by replacing special characters with their HTML entity equivalents: 

  • < becomes <
  • > becomes >
  • & becomes &
