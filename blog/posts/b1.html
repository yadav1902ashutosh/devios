How to Display HTML Code as Text in a Webpage

Published on February 14, 2025 | By Devios

When writing a blog or documentation about web development, you might want to display HTML code as plain text rather than letting the browser interpret it. In this article, we'll explore different ways to achieve this.

Why Does HTML Render Instead of Displaying as Text?

Browsers are designed to interpret HTML tags and render them accordingly. For example, writing:

<p>This is a paragraph.</p>

Will result in the browser displaying:

This is a paragraph.

Instead of the actual HTML code. To prevent this, we need to explicitly tell the browser to treat HTML as plain text.

Methods to Display HTML Code as Text

1. Using HTML Entities

One simple way to display HTML code is by replacing special characters with their HTML entity equivalents:

  • < becomes &lt;
  • > becomes &gt;
  • & becomes &amp;

For example, writing:

&lt;p&gt;This is a paragraph.&lt;/p&gt;

Will display:

This is a paragraph.

2. Using the <pre> Tag

For better readability, you can use the <pre> tag along with the <code> tag:

                
<pre>
    <p>This is a paragraph.</p>
</pre>

This will preserve indentation and make the code block more readable.

3. Using JavaScript to Escape HTML

If you're dynamically inserting content, JavaScript can help escape HTML characters:

                
function escapeHTML(html) {
    return html.replace(//g, ">");
}

document.getElementById("output").innerHTML = escapeHTML("<p>This is a paragraph.</p>");

4. Using CSS to Prevent Interpretation

You can also use CSS to prevent the browser from rendering HTML inside certain elements:


pre, code {
    white-space: pre;
    font-family: monospace;
}

Conclusion

Displaying HTML code as text is essential when writing tutorials or documentation about web development. By using the methods mentioned above, you can ensure that your code snippets are displayed correctly and are easy to read.

Using HTML entities is the simplest way, but combining <pre> and <code> provides better readability. For dynamic content, JavaScript can be used to escape HTML automatically.
